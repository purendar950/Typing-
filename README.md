# StudyPlanner Typing

A complete typing test platform for government exam typing practice (DSSSB JSA/LDC), built using the same resources as EzTyping.

## Features

- **42 DSSSB Typing Passages** (41 English + 1 Hindi)
- **10-minute timed tests** with auto-submit
- **Live WPM tracking** (Net & Gross speed)
- **Real-time accuracy** with color-coded word highlighting
- **Detailed error analysis** showing exactly which words were wrong
- **Mobile responsive** design
- **Paste protection** during tests
- **Test selection grid** with language filtering

## How to Use

1. Open `index.html` in a browser
2. Select a test from the grid
3. Click "Start Test" and begin typing
4. Results shown on completion or when time runs out

## Source

Passages downloaded from eztyping.com DSSSB JSA typing test category.
Internal IDs: 6415-6537 (English), 221 (Hindi)
