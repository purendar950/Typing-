# StudyPlanner Typing - DSSSB JSA Typing Test Practice

Complete typing test practice platform with **247 tests** (240 English + 7 Hindi) for DSSSB JSA/LDC skill test preparation.

## Features
- 240 English typing tests (External #1-240)
- 7 Hindi typing tests  
- 10-minute timed tests
- Real-time WPM, accuracy tracking
- Error highlighting and summary
- Mobile-responsive design
- Keyboard layout support (Inscript for Hindi)

## Tests Coverage
- All DSSSB JSA typing test passages
- Based on actual exam patterns
- Sourced from eztyping.com

## Usage
Open index.html in any browser, or deploy to GitHub Pages.
