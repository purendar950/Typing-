# StudyPlanner Typing - DSSSB JSA Typing Test Practice

**461 typing tests** (240 English + 221 Hindi) for DSSSB JSA/LDC skill test preparation.

## Features
- **240 English typing tests** (External #1-240) - Complete set!
- **221 Hindi typing tests** (External #1-221) 
- 10-minute timed tests with real-time WPM & accuracy tracking
- Error highlighting and summary
- Mobile-responsive design
- Multiple keyboard layouts (Remington GAIL, Inscript for Hindi)

## Tests Coverage
- All DSSSB JSA typing test passages sourced from eztyping.com
- Covers all exam patterns and difficulty levels
- Regular updates as new tests become available

## Usage
Open `index.html` in any browser, or deploy to GitHub Pages.

## GitHub Pages
The repo includes a GitHub Actions workflow for automatic Pages deployment.
Enable in: Settings → Pages → Deploy from main branch.
